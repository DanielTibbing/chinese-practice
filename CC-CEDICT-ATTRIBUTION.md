# CC-CEDICT Attribution

This project ships dictionary data derived from CC-CEDICT via the npm package
`@tykok/cedict-dictionary`.

Sources:

- https://www.npmjs.com/package/@tykok/cedict-dictionary
- https://www.mdbg.net/chinese/dictionary?page=cc-cedict
- https://cc-cedict.org/wiki/start

CC-CEDICT is published for reuse with attribution and share-alike terms. If you
publish this site, keep this attribution file with the generated output.
