How to add study material

Flashcards

Create or update data/flashcards.json.

{
  "decks": [
    {
      "id": "deck-id",
      "name": "Deck name",
      "cards": [
        {
          "hanzi": "你好",
          "pinyin": "ni hao",
          "translation": "hello",
          "notes": "optional"
        }
      ]
    }
  ]
}

Texts and Lyrics

Add lyrics as one file per song in content/lyrics/ and general readings in content/texts/. 

{
  "texts": [
    {
      "id": "song-id",
      "title": "Title",
      "description": "Optional",
      "tags": ["lyrics"],
      "lines": [
        {
          "hanzi": "月亮代表我的心",
          "pinyin": "yue liang dai biao wo de xin",
          "translation": "The moon represents my heart"
        }
      ]
    }
  ]
}

Dictionary

Create or update data/dictionary.json.

{
  "entries": [
    {
      "traditional": "學習",
      "simplified": "学习",
      "pinyin": "xue xi",
      "definitions": ["to learn", "to study"],
      "hsk": 1
    }
  ]
}
